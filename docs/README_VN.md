# Tài liệu Dự án

Chào mừng bạn đến với thư mục tài liệu trung tâm của dự án Ecommerce BookStore.

## Mục lục

- [Kiến trúc Hệ thống & CI/CD Pipeline](./architecture-and-cicd-vn.md): Tổng quan chi tiết về kiến trúc kỹ thuật của dự án, sự tương tác giữa các thành phần và quy trình tích hợp/triển khai liên tục (CI/CD).
- [Hướng dẫn Deploy lên Render.com](./render-deployment-guide-vn.md): Hướng dẫn từng bước cách triển khai dự án này lên Render.com bằng IaC Blueprint và GitHub Actions deploy hook.
- **`portfolio/`**: Chứa các ảnh chụp màn hình và tài nguyên được sử dụng để giới thiệu dự án trong Portfolio.
